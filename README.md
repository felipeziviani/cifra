# CIFRA

CIFRA é um projeto web desenvolvido com HTML e CSS puro que funciona como um guia de lugares "escondidos" e alternativos na cidade.

A proposta é simular um sistema antigo / arquivo secreto, utilizando o estilo **Brutalismo Digital**.

---

## Objetivo

Criar uma experiência diferente dos sites tradicionais, com foco em:

- Exploração urbana
- Lugares underground
- Estética minimalista e brutalista

---

## Público-Alvo

Jovens e pessoas interessadas em:

- Lugares alternativos
- Experiências únicas
- Cultura fora do mainstream

## Estilo Visual

- Cores: Preto, Branco e Laranja
- Tipografia estilo terminal / máquina de escrever
- Layout com aparência de sistema antigo

- Home → Lista de lugares

- Sobre (Protocolo) → Regras + formulário
